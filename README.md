# FinanceCLI — консольный менеджер финансов

**FinanceCLI** — это консольное Java-приложение для управления личными финансами.

Программа позволяет:

- Добавлять доходы и расходы
- Управлять бюджетами по категориям
- Смотреть статистику (полную и по категориям/временному периоду)
- Переводить деньги другим пользователям
- Импортировать и экспортировать данные в JSON
- Удалять пользователя вместе со всеми данными

---

## Запуск проекта

### Требования

- Java **17+**
- Maven **3.8+**

### Сборка и запуск

```sh
mvn clean package
java -jar target/FinanceCLI.jar
```

Или через Maven:

```bash
mvn exec:java -Dexec.mainClass="art.FinanceControl"
```

### Запуск тестов

```sh
mvn test
```

## Принцип работы

| Слой | Что делает |
|------|------------|
| `model` | DTO + основные структуры данных |
| `service` | Чистая бизнес-логика |
| `port` | Абстракции для взаимодействия |
| `infra` | Реальные реализации (JSON-хранилище и т.д.) |
| `cli` | UI над логикой (меню, примеры, help) |

## Примеры команд 

### Добавление дохода

```
Пример добавления дохода:
1. Перейдите в «Управление транзакциями».
2. Выберите пункт «Добавить доход».
3. Введите категорию, например: "Зарплата".
4. Введите сумму, например: 1000.
Результат: создана новая транзакция — доход 1000 в категории "Зарплата".
```

### ➖ Добавление расхода

```
Пример добавления расхода:
1. Откройте меню «Управление транзакциями».
2. Выберите пункт «Добавить расход».
3. Укажите категорию, например: "Продукты".
4. Введите сумму, например: 350.
Результат: создан расход 350 в категории "Продукты".
```

### Изменение транзакции

```
1. Выберите «Изменить транзакцию»
2. Выберите номер в списке
3. Можно изменить категорию, сумму или тип (доход/расход)
```

### Статистика по категориям

```
Введите дату от: 2024.01.01 00:00:00
Введите дату до:  2024.02.01 00:00:00
Категории: Еда, Путешествия
Результат: вывод статистики только по выбранным категориям
```
